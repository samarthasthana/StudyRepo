
Q1
design and implement a calculater that can calculate expressions like:
  + 2 4
  * 8 ( + 7 12)
  ( + 7 ( * 8 12 ) ( * 2 (+ 9 4) 7 ) 3 )

  (PS:all items are space delimetered.)

  Example answers
  + 2 4 => 2 + 4 = 6
  * 8 ( + 7 12) => 8 * ( 7 + 12 ) = 152
  ( + 7 ( * 8 12 ) ( * 2 (+ 9 4) 7 ) 3 ) => 7+8*12+2*(9+4)*7+3 = 148
